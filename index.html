Welcome to my CM124 Final Project

This project is designed to assemble complete haplotypes from a series of incomplete reads. Humans have two haplotypes, one from each parent. Unfortunately for us, sequencing reads done by machine only contain information regarding one of these haplotypes. Information on both haplotypes is contained within this sequencing read, however. So, by tiling together reads (using a greedy algorithm written in Python) we can reassemble complete haplotypes.

Goals

  • Come up with an algorithm to go through reads and place them in the haplotype they match
  • Write a function to generate data
  • Write a function to read this data into strings my algorithm can work with
  • Assemble complete haplotypes assuming no errors
